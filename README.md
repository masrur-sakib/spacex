## SpaceX Project:

### Installation Steps:

Open terminal in your IDE/Code Editor and run the below commands ony by one,

- First Command: `git clone https://github.com/masrur-sakib/spacex-project.git`
- Second Command: `npm install`
- Third Command: `npm start`
